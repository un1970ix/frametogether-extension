## Build

```bash
# Install dependencies.
bun install

# Build for Chrome.
bun run build

# Build for Firefox.
bun run build:firefox
```

## Development

```bash
# Run this command to start in watch mode.
bun run dev
```

Load the `dist/` directory as unpacked extension.
